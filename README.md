# Zippily llms.txt

This repository hosts the llms.txt file for Zippily (www.zippily.co.nz).

The llms.txt file helps AI tools like ChatGPT, Perplexity, and Claude understand 
what Zippily does and index the website accurately for AI-driven search.

## What is llms.txt?
llms.txt is an emerging standard that gives AI crawlers a structured, plain-text 
summary of a website's content, services, and key pages.
